## 價格配置邏輯

### 服務類型映射
- **債務糾紛** → 咨詢價格 (`consultation_base_price`)
- **家庭法律** → 咨詢價格
- **商業法律** → 咨詢價格
- **房產法律** → 咨詢價格
- **刑事辯護** → 咨詢價格
- **勞動法律** → 審核價格 (`review_base_price`)
- **知識產權** → 咨詢價格

### 默認價格（API 失敗時）
- 咨詢服務：RM 150
- 審核服務：RM 300

## 測試步驟

### 1. 在 Admin 後台設置價格
1. 訪問：`http://localhost:3000/admin/pricing`
2. 設置價格：
   - **貨幣設置**：選擇 `馬來西亞令吉 (MYR)`
   - **基礎咨詢**：設置為 `30` MYR（時長：`30` 分鐘）
   - **高級咨詢**：設置為 `120` MYR（時長：`120` 分鐘）
   - **標準審核**：設置為 `60` MYR（時長：`標準審核服務`）
3. 點擊「保存修改」

### 2. 檢查前台價格
1. 訪問：`http://localhost:3000`
2. 滾動到「熱門服務」區域
3. 應該看到：
   - **債務糾紛**：RM 30
   - **家庭法律**：RM 30
   - **商業法律**：RM 30
   - **房產法律**：RM 30
   - **刑事辯護**：RM 30
   - **勞動法律**：RM 60（審核價格）
   - **知識產權**：RM 30

### 3. 測試貨幣切換
1. 在 Admin 後台切換貨幣為 `美元 (USD)`
2. 設置價格：
   - 基礎咨詢：`10` USD
   - 標準審核：`20` USD
3. 保存後刷新前台頁面
4. 應該看到：
   - **債務糾紛**：$ 10
   - **勞動法律**：$ 20

### 4. 測試 API 失敗情況
1. 暫時停止數據庫或 API
2. 刷新前台頁面
3. 應該看到默認價格：
   - 咨詢服務：RM 150
   - 審核服務：RM 300

## API 端點

### 公開 API（不需要登錄）
- **URL**: `/api/public/tenant-config`
- **方法**: GET
- **返回**: 租戶的所有配置（品牌、價格、功能等）
- **緩存**: 5 分鐘（`s-maxage=300`）

### 返回數據格式
```json
{
  "tenant": {
    "id": "uuid",
    "name": "租戶名稱",
    "subdomain": "subdomain"
  },
  "pricing": {
    "currency": "MYR",
    "consultation_base_price": 30,
    "consultation_duration": 30,
    "review_base_price": 60,
    "review_description": "標準審核服務"
  },
  "branding": { ... },
  "features": { ... }
}
```

## 注意事項

1. **緩存問題**
   - API 有 5 分鐘緩存
   - 如果修改價格後前台沒有立即更新，請等待 5 分鐘或強制刷新（Ctrl + Shift + R）

2. **價格格式化**
   - 使用 `toLocaleString()` 格式化數字（例如：1000 → 1,000）
   - 貨幣符號根據配置的貨幣類型自動顯示

3. **錯誤處理**
   - 如果 API 失敗，組件會使用默認價格
   - 不會導致頁面崩潰或顯示錯誤
