## 🌐 部署步骤（快速版）

### 1. 登录 Vercel
访问：https://vercel.com
- 点击 "Sign Up" 或 "Log In"
- 选择 "Continue with GitHub"
- 授权 Vercel

### 2. 导入项目
- 点击 "Add New..." → "Project"
- 选择 `yuyue123yu/yyu` 仓库
- 点击 "Import"

### 3. 添加环境变量
点击 "Environment Variables"，添加 5 个变量：

```
NEXT_PUBLIC_SUPABASE_URL=https://xxx.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=eyJxxx...
SUPABASE_SERVICE_ROLE_KEY=eyJxxx...
NEXT_PUBLIC_SITE_URL=https://yyu.vercel.app
NODE_ENV=production
```

**从 Supabase 获取前 3 个变量：**
1. 登录 https://supabase.com
2. 选择您的项目
3. 进入 Settings → API
4. 复制 URL 和 Keys

### 4. 部署
- 点击 "Deploy"
- 等待 2-3 分钟

### 5. 更新域名
- 复制实际域名（如 `https://yyu-abc123.vercel.app`）
- 在 Settings → Environment Variables 中更新 `NEXT_PUBLIC_SITE_URL`
- 重新部署
