## 🔧 常见问题

### Q: 页面显示"需要登录"？
**A**: 请先访问 http://localhost:3000/admin/login 登录

### Q: 保存后没有反应？
**A**: 
1. 检查浏览器控制台是否有错误
2. 检查网络请求是否成功
3. 刷新页面查看配置是否保存

### Q: 某个功能不工作？
**A**: 
1. 打开浏览器开发者工具（F12）
2. 查看 Console 标签是否有错误
3. 查看 Network 标签查看 API 请求
4. 记录错误信息
