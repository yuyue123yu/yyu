# 🏷️ 品牌名修改指南

## 📋 快速修改方法

### 方法一：修改品牌配置文件（推荐）⭐

**文件位置：** `src/config/brand.ts`

只需修改这一个文件，然后在其他地方引用即可：

```typescript
export const BRAND_CONFIG = {
  // 修改这里的品牌名称
  name: 'LegalMY',  // 改成您想要的品牌名
  
  // 修改公司全称
  companyName: 'LegalMY Sdn Bhd',  // 改成您的公司名
  
  // 修改联系邮箱
  contact: {
    email: {
      info: 'info@legalmy.com',  // 改成您的邮箱
      support: 'support@legalmy.com',
      // ... 其他邮箱
    },
    phone: '+60 3-1234 5678',  // 改成您的电话
    address: 'Level 10, Menara LegalMY...',  // 改成您的地址
  },
};
```

### 方法二：全局搜索替换

使用VS Code的全局搜索替换功能：

1. 按 `Ctrl + Shift + H` 打开搜索替换
2. 搜索：`LegalMY`
3. 替换为：`您的品牌名`
4. 点击"全部替换"

⚠️ **注意：** 这个方法会替换所有出现的地方，包括文件名、注释等。

## ⚠️ 注意事项

### 1. 邮箱域名
如果您修改了品牌名，记得同时修改邮箱域名：
- `info@legalmy.com` → `info@yourbrand.com`
- `support@legalmy.com` → `support@yourbrand.com`

### 2. 公司注册信息
修改以下法律信息：
- 公司全称
- 注册号
- 公司地址

### 3. Logo和Favicon
如果有Logo文件，也需要替换：
- `/public/logo.png`
- `/public/favicon.ico`

### 4. 翻译文件
如果使用了多语言，记得修改翻译文件：
- `src/contexts/LanguageContext.tsx`

### 5. 数据库配置
如果数据库中存储了品牌信息，也需要更新：
- `system_settings` 表
- `tenant_settings` 表
