## ⚠️ 注意事项

### 关于 RLS 警告

如果看到这个警告：
```
New table will not have Row Level Security enabled
```

**不用担心！** 这是正常的。选择：
- **"Run and enable RLS"**（绿色按钮）- 推荐
- 或 "Run without RLS"（橙色按钮）- 然后手动启用

### 如果执行失败

1. **检查错误信息**
   - 如果是 "already exists"，说明已经执行过了，可以跳过
   - 如果是其他错误，复制错误信息

2. **回滚方案**
   - 索引可以安全删除：`DROP INDEX IF EXISTS idx_xxx;`
   - 字段可以删除：`ALTER TABLE xxx DROP COLUMN IF EXISTS deleted_at;`

3. **重新执行**
   - 所有 SQL 都使用了 `IF NOT EXISTS`
   - 可以安全地重复执行
