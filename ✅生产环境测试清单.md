### 6️⃣ 安全性测试

#### 6.1 认证和授权
- [ ] **未登录访问保护**
  - 尝试直接访问 `/admin`
  - 应该重定向到登录页
  
- [ ] **权限隔离**
  - 普通用户尝试访问 `/super-admin`
  - 应该显示 403 或重定向
  
- [ ] **会话管理**
  - 登录后刷新页面
  - 会话应该保持
  - 退出登录后无法访问

#### 6.2 数据安全
- [ ] **RLS 策略**
  - 租户 A 的管理员无法看到租户 B 的数据
  - 使用不同账号测试
  
- [ ] **SQL 注入防护**
  - 在表单中输入 SQL 语句
  - 应该被正确转义
  
- [ ] **XSS 防护**
  - 在表单中输入 `<script>alert('XSS')</script>`
  - 应该被正确转义

#### 6.3 HTTPS 和 Headers
- [ ] **HTTPS 强制**
  - 检查是否强制使用 HTTPS
  
- [ ] **安全 Headers**
  - 使用浏览器 DevTools 检查响应头
  - 应该包含: CSP, X-Frame-Options, X-Content-Type-Options
