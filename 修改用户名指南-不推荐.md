## 🛠️ 如果您坚持要修改

### 方法 1：创建新用户（推荐）

这是最安全的方法：

1. **创建新用户**
   - 按 `Win + I` 打开设置
   - 进入 "账户" → "家庭和其他用户"
   - 点击 "添加其他用户"
   - 创建用户名为 "yu" 的新账户
   - 设置为管理员

2. **迁移数据**
   - 登录新账户
   - 从 `C:\Users\余\` 复制文件到 `C:\Users\yu\`
   - 重新安装必要的软件

3. **删除旧账户**
   - 确认新账户一切正常
   - 删除 "余" 账户

### 方法 2：修改注册表（高风险）

**警告：此方法可能导致系统无法启动！**

1. **备份注册表**
   ```powershell
   reg export "HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\ProfileList" C:\backup_registry.reg
   ```

2. **修改用户文件夹名称**
   - 创建新管理员账户
   - 登录新账户
   - 重命名 `C:\Users\余` 为 `C:\Users\yu`

3. **修改注册表**
   - 按 `Win + R`，输入 `regedit`
   - 导航到：`HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\ProfileList`
   - 找到您的用户 SID
   - 修改 `ProfileImagePath` 为 `C:\Users\yu`

4. **重启电脑**
