# Logo 显示位置说明

## 📍 您上传的 Logo 会在以下位置显示

### 1. **首页顶部导航栏（Header）**
- **位置**：页面最顶部左侧
- **大小**：高度 40px，宽度自适应
- **功能**：点击可返回首页
- **显示逻辑**：
  - 如果上传了 Logo → 显示 Logo 图片
  - 如果没有上传 → 显示图标 + 网站名称

### 2. **首页底部页脚（Footer）**
- **位置**：页面底部左侧
- **大小**：高度 32px，宽度自适应
- **样式**：白色 Logo（使用 CSS 滤镜反色，适配深色背景）
- **显示逻辑**：
  - 如果上传了 Logo → 显示 Logo 图片
  - 如果没有上传 → 显示图标 + 网站名称

## 🎨 Logo 上传建议

### 推荐格式
- **PNG**（支持透明背景，推荐）
- **SVG**（矢量图，最佳选择）
- **JPG**（不支持透明背景）
- **WebP**（现代格式，体积小）

### 推荐尺寸
- **宽度**：300-600px
- **高度**：80-120px
- **比例**：横向 Logo（宽 > 高）效果最佳
- **文件大小**：建议 < 200KB

### 设计建议
1. **简洁明了**：Logo 应该清晰易识别
2. **透明背景**：使用 PNG 格式，背景透明
3. **高对比度**：确保在白色和深色背景上都清晰可见
4. **横向布局**：横向 Logo 在导航栏中显示效果更好

## 🔄 如何更新 Logo

1. 登录 Admin 后台：`http://localhost:3000/admin`
2. 点击左侧菜单「品牌设置」
3. 在「Logo 设置」区域点击「上传 Logo」
4. 选择您的 Logo 文件
5. 上传成功后，刷新首页即可看到新 Logo

## 📱 响应式显示

- **桌面端**：Logo 完整显示
- **平板端**：Logo 完整显示
- **手机端**：Logo 完整显示（导航栏会自动调整）

## 🎯 实时预览

修改 Logo 后：
1. 首页 Header 会立即显示新 Logo
2. 首页 Footer 会立即显示新 Logo（白色版本）
3. 所有页面的导航栏都会更新

## 💡 技术实现

Logo 通过以下方式集成：
- 存储在 Supabase Storage 的 `tenant-assets` bucket
- 通过 `SiteSettingsContext` 全局状态管理
- 自动从品牌设置 API 读取
- 支持多租户隔离（每个租户有独立的 Logo）

## 🔍 调试

如果 Logo 没有显示：
1. 检查浏览器控制台是否有错误
2. 确认 Logo 已成功上传到 Supabase Storage
3. 检查 `settings.logoUrl` 是否有值
4. 刷新页面清除缓存
