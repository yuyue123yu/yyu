## 📋 功能概述

内容管理功能允许租户自主编辑网站的文字内容和描述，包括：

1. **首页横幅** - 主标题、副标题、描述、按钮、背景图
2. **服务介绍** - 服务标题、描述、图标
3. **关于我们** - 公司介绍、使命、愿景、统计数据
4. **常见问题** - FAQ 问答列表
5. **页脚内容** - 公司简介、联系信息、版权信息

## 🔧 技术实现

### API 路由

**GET `/api/tenant/content`**
- 获取租户的内容配置
- 返回默认值或已保存的设置
- 权限：已登录用户

**PUT `/api/tenant/content`**
- 更新租户的内容配置
- 权限：Owner, Admin
- 记录审计日志

### 数据存储

存储在 `tenant_settings` 表：
```json
{
  "tenant_id": "uuid",
  "key": "content",
  "value": {
    "hero": { ... },
    "services": { ... },
    "about": { ... },
    "faq": { ... },
    "footer": { ... }
  }
}
```

### 前端组件

**页面**: `src/app/admin/content/page.tsx`
- 标签式导航
- 表单编辑
- 动态列表管理
- 实时状态更新

## 🚀 使用流程

### 1. 访问内容管理页面
```
/admin/content
```

### 2. 选择要编辑的部分
- 点击顶部标签切换不同部分
- 首页横幅、服务介绍、关于我们、常见问题、页脚内容

### 3. 编辑内容
- 在表单中输入或修改内容
- 对于列表项，可以添加或删除
- 所有修改实时保存在本地状态

### 4. 保存设置
- 点击右上角"保存内容"按钮
- 等待保存成功提示
- 内容立即在前台生效

### 5. 查看效果
- 访问前台网站
- 查看修改后的内容
- 确认显示正确

## 💡 使用建议

### 内容编写技巧

1. **首页横幅**
   - 标题简洁有力（5-10 字）
   - 副标题说明价值（10-20 字）
   - 描述详细但不冗长（50-100 字）

2. **服务介绍**
   - 每个服务标题清晰（3-5 字）
   - 描述突出特点和优势（20-30 字）
   - 选择合适的图标

3. **关于我们**
   - 内容真实可信
   - 突出专业性和经验
   - 使用具体数据增强说服力

4. **常见问题**
   - 问题简洁明了
   - 答案详细但易懂
   - 覆盖用户最关心的问题

5. **页脚内容**
   - 联系信息准确完整

## 🎯 前台应用示例

### 获取内容配置
```typescript
// 在前台页面获取内容
const { content } = await getTenantConfig(tenantId);

// 使用首页横幅内容
<section className="hero">
  <h1>{content.hero.title}</h1>
  <p>{content.hero.subtitle}</p>
  <p>{content.hero.description}</p>
  <a href={content.hero.cta_link}>
    {content.hero.cta_text}
  </a>
</section>

// 使用服务介绍
<section className="services">
  <h2>{content.services.title}</h2>
  <p>{content.services.subtitle}</p>
  <div className="service-grid">
    {content.services.items.map(service => (
      <div key={service.title}>
        <Icon name={service.icon} />
        <h3>{service.title}</h3>
        <p>{service.description}</p>
      </div>
    ))}
  </div>
</section>

// 使用关于我们
<section className="about">
  <h2>{content.about.title}</h2>
  <p>{content.about.content}</p>
  <div className="stats">
    <div>团队规模: {content.about.team_size}</div>
    <div>从业年限: {content.about.years_experience}</div>
    <div>处理案件: {content.about.cases_handled}</div>
    <div>客户满意度: {content.about.client_satisfaction}</div>
  </div>
</section>

// 使用 FAQ
<section className="faq">
  <h2>{content.faq.title}</h2>
  {content.faq.items.map((item, index) => (
    <div key={index}>
      <h3>{item.question}</h3>
      <p>{item.answer}</p>
    </div>
  ))}
</section>

// 使用页脚
<footer>
  <p>{content.footer.company_description}</p>
  <div>
    <p>{content.footer.contact.address}</p>
    <p>{content.footer.contact.phone}</p>
    <p>{content.footer.contact.email}</p>
    <p>{content.footer.contact.business_hours}</p>
  </div>
  <p>{content.footer.copyright}</p>
</footer>
```
