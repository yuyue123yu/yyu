# 账号管理系统 - 完整方案

**适用范围**：Super Admin 系统 + 租户系统  
**核心功能**：子账号管理、密码修改、密码重置

---

## 📋 需求分析

### 1. 租户系统需求
- ✅ **子账号管理**：租户管理员可以创建多个子账号
- ✅ **角色权限**：主账号、管理员、普通员工
- ✅ **密码修改**：用户可以修改自己的密码
- ✅ **忘记密码**：通过邮件重置密码

### 2. Super Admin 系统需求
- ✅ **密码修改**：Super Admin 可以修改自己的密码
- ✅ **忘记密码**：通过邮件重置密码
- ✅ **管理员账号管理**：创建/管理其他 Super Admin

## 🎯 功能设计

### 功能 1：子账号管理

#### 1.1 创建子账号

**权限要求**：
- 租户系统：owner 或 admin 角色
- Super Admin 系统：super_admin = true

**流程**：
```
1. 管理员填写子账号信息（邮箱、姓名、角色）
2. 系统创建 Supabase Auth 用户（随机密码）
3. 创建 profile 记录（关联 parent_user_id 和 tenant_id）
4. 发送欢迎邮件（包含设置密码链接）
5. 记录审计日志
```

**API 设计**：
```typescript
// POST /api/tenant/users/create
{
  email: string;
  full_name: string;
  role: 'admin' | 'manager' | 'user';
  permissions?: object;
}

// Response
{
  success: boolean;
  user: {
    id: string;
    email: string;
    full_name: string;
    role: string;
  };
  message: string;
}
```

#### 1.2 子账号列表

**显示内容**：
- 邮箱、姓名、角色
- 创建时间、最后登录时间
- 状态（激活/禁用）
- 操作按钮（编辑、禁用、删除）

**筛选功能**：
- 按角色筛选
- 按状态筛选
- 搜索（邮箱、姓名）

#### 1.3 编辑子账号

**可编辑内容**：
- 姓名
- 角色
- 权限配置
- 激活状态

**不可编辑**：
- 邮箱（Supabase Auth 限制）
- 创建时间

#### 1.4 禁用/启用子账号

**禁用效果**：
- 设置 `is_active = false`
- 用户无法登录（在登录时检查）
- 现有 session 保持有效（可选：强制登出）

**启用效果**：
- 设置 `is_active = true`
- 用户可以正常登录

#### 1.5 删除子账号

**删除流程**：
1. 确认对话框（防止误删）
2. 删除 profile 记录
3. 删除 Supabase Auth 用户
4. 记录审计日志

**注意**：
- 主账号（owner）不能被删除
- 删除后无法恢复

### 功能 2：密码修改

#### 2.1 用户自己修改密码

**入口**：
- 用户设置页面
- 个人资料页面

**流程**：
```
1. 用户输入当前密码
2. 用户输入新密码（两次确认）
3. 验证当前密码是否正确
4. 更新密码
5. 发送通知邮件
6. 记录密码修改历史
```

**API 设计**：
```typescript
// POST /api/auth/change-password
{
  currentPassword: string;
  newPassword: string;
  confirmPassword: string;
}

// Response
{
  success: boolean;
  message: string;
}
```

**密码要求**：
- 最少 8 个字符
- 至少包含 1 个大写字母
- 至少包含 1 个小写字母
- 至少包含 1 个数字
- 至少包含 1 个特殊字符（可选）

#### 2.2 管理员重置用户密码

**权限要求**：
- 租户系统：owner 或 admin
- Super Admin 系统：super_admin = true

**流程**：
```
1. 管理员选择要重置的用户
2. 系统生成临时密码或发送重置邮件
3. 记录密码重置历史
4. 发送通知邮件给用户
```

**两种方式**：

**方式 1：生成临时密码**
```typescript

{
  method: 'temporary'
}

// Response
{
  success: boolean;
  temporaryPassword: string; // 仅显示一次
  message: string;
}
```

**方式 2：发送重置邮件**
```typescript

{
  method: 'email'
}

// Response
{
  success: boolean;
  message: '重置邮件已发送';
}
```

## 🔧 技术实现

### 1. 创建子账号 API

**文件**：`src/app/api/tenant/users/create/route.ts`

```typescript
import { NextRequest, NextResponse } from 'next/server';
import { createServerClient } from '@/lib/supabase/server';

export async function POST(request: NextRequest) {
  try {
    const supabase = await createServerClient();
    
    // 1. 验证当前用户权限
    const { data: { session } } = await supabase.auth.getSession();
    if (!session) {
      return NextResponse.json({ error: 'Unauthorized' }, { status: 401 });
    }

    const { data: profile } = await supabase
      .from('profiles')
      .select('tenant_id, role, super_admin')
      .eq('id', session.user.id)
      .single();

    if (!profile || (profile.role !== 'owner' && profile.role !== 'admin' && !profile.super_admin)) {
      return NextResponse.json({ error: 'Forbidden' }, { status: 403 });
    }

    // 2. 获取请求数据
    const { email, full_name, role, permissions } = await request.json();

    // 3. 验证数据
    if (!email || !full_name || !role) {
      return NextResponse.json({ error: 'Missing required fields' }, { status: 400 });
    }

    // 4. 生成随机密码
    const temporaryPassword = generateRandomPassword();

    // 5. 创建 Supabase Auth 用户
    const { data: authUser, error: authError } = await supabase.auth.admin.createUser({
      email,
      password: temporaryPassword,
      email_confirm: true, // 自动确认邮箱
      user_metadata: {
        full_name,
      },
    });

    if (authError || !authUser.user) {
      return NextResponse.json({ error: authError?.message || 'Failed to create user' }, { status: 500 });
    }

    // 6. 创建 profile 记录
    const { error: profileError } = await supabase
      .from('profiles')
      .insert({
        id: authUser.user.id,
        email,
        full_name,
        user_type: 'user',
        role,
        tenant_id: profile.tenant_id,
        parent_user_id: session.user.id,
        permissions: permissions || {},
        is_active: true,
      });

    if (profileError) {
      // 回滚：删除 Auth 用户
      await supabase.auth.admin.deleteUser(authUser.user.id);
      return NextResponse.json({ error: 'Failed to create profile' }, { status: 500 });
    }

    // 7. 发送欢迎邮件（包含设置密码链接）
    await sendWelcomeEmail(email, full_name, temporaryPassword);

    // 8. 记录审计日志
    await supabase.from('audit_logs').insert({
      user_id: session.user.id,
      action: 'create_sub_account',
      resource_type: 'user',
      resource_id: authUser.user.id,
      details: { email, full_name, role },
    });

    return NextResponse.json({
      success: true,
      user: {
        id: authUser.user.id,
        email,
        full_name,
        role,
      },
      message: '子账号创建成功，欢迎邮件已发送',
    });
  } catch (error: any) {
    console.error('Error creating sub-account:', error);
    return NextResponse.json({
      success: false,
      error: error.message,
    }, { status: 500 });
  }
}

function generateRandomPassword(length: number = 12): string {
  const charset = 'abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ0123456789!@#$%^&*';
  let password = '';
  for (let i = 0; i < length; i++) {
    password += charset.charAt(Math.floor(Math.random() * charset.length));
  }
  return password;
}
```

### 2. 修改密码 API

**文件**：`src/app/api/auth/change-password/route.ts`

```typescript
import { NextRequest, NextResponse } from 'next/server';
import { createServerClient } from '@/lib/supabase/server';

export async function POST(request: NextRequest) {
  try {
    const supabase = await createServerClient();
    
    // 1. 验证用户登录
    const { data: { session } } = await supabase.auth.getSession();
    if (!session) {
      return NextResponse.json({ error: 'Unauthorized' }, { status: 401 });
    }

    // 2. 获取请求数据
    const { currentPassword, newPassword, confirmPassword } = await request.json();

    // 3. 验证数据
    if (!currentPassword || !newPassword || !confirmPassword) {
      return NextResponse.json({ error: 'Missing required fields' }, { status: 400 });
    }

    if (newPassword !== confirmPassword) {
      return NextResponse.json({ error: '两次输入的密码不一致' }, { status: 400 });
    }

    // 4. 验证密码强度
    if (!isPasswordStrong(newPassword)) {
      return NextResponse.json({ 
        error: '密码强度不足：至少8个字符，包含大小写字母和数字' 
      }, { status: 400 });
    }

    // 5. 验证当前密码
    const { error: signInError } = await supabase.auth.signInWithPassword({
      email: session.user.email!,
      password: currentPassword,
    });

    if (signInError) {
      return NextResponse.json({ error: '当前密码不正确' }, { status: 400 });
    }

    // 6. 更新密码
    const { error: updateError } = await supabase.auth.updateUser({
      password: newPassword,
    });

    if (updateError) {
      return NextResponse.json({ error: updateError.message }, { status: 500 });
    }

    // 7. 记录密码修改历史
    await supabase.from('password_reset_history').insert({
      user_id: session.user.id,
      reset_method: 'self',
      ip_address: request.headers.get('x-forwarded-for') || request.headers.get('x-real-ip'),
      user_agent: request.headers.get('user-agent'),
    });

    // 8. 发送通知邮件
    // TODO: 发送密码修改通知邮件

    return NextResponse.json({
      success: true,
      message: '密码修改成功',
    });
  } catch (error: any) {
    console.error('Error changing password:', error);
    return NextResponse.json({
      success: false,
      error: error.message,
    }, { status: 500 });
  }
}

function isPasswordStrong(password: string): boolean {
  // 至少8个字符
  if (password.length < 8) return false;
  
  // 包含大写字母
  if (!/[A-Z]/.test(password)) return false;
  
  // 包含小写字母
  if (!/[a-z]/.test(password)) return false;
  
  // 包含数字
  if (!/[0-9]/.test(password)) return false;
  
  return true;
}
```

### 3. 忘记密码 API

**文件**：`src/app/api/auth/forgot-password/route.ts`

```typescript
import { NextRequest, NextResponse } from 'next/server';
import { createClient } from '@supabase/supabase-js';

export async function POST(request: NextRequest) {
  try {
    // 使用 service role key 来发送重置邮件
    const supabase = createClient(
      process.env.NEXT_PUBLIC_SUPABASE_URL!,
      process.env.SUPABASE_SERVICE_ROLE_KEY!
    );

    // 1. 获取请求数据
    const { email } = await request.json();

    // 2. 验证邮箱格式
    if (!email || !isValidEmail(email)) {
      return NextResponse.json({ error: 'Invalid email' }, { status: 400 });
    }

    // 3. 检查请求频率（防止滥用）
    // TODO: 实现 rate limiting

    // 4. 发送重置邮件
    const { error } = await supabase.auth.resetPasswordForEmail(email, {
      redirectTo: `${process.env.NEXT_PUBLIC_APP_URL}/reset-password`,
    });

    // 5. 不管邮箱是否存在，都返回成功（安全考虑）
    return NextResponse.json({
      success: true,
      message: '如果该邮箱存在，重置邮件已发送',
    });
  } catch (error: any) {
    console.error('Error sending reset email:', error);
    return NextResponse.json({
      success: false,
      error: 'Failed to send reset email',
    }, { status: 500 });
  }
}

function isValidEmail(email: string): boolean {
  return /^[^\s@]+@[^\s@]+\.[^\s@]+$/.test(email);
}
```
