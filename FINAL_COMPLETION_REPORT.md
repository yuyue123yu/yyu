## 📝 使用指南

### 用户使用
1. 访问网站任意页面
2. 点击右上角的语言切换器
3. 选择 中文 / English / Bahasa Malaysia
4. 所有核心页面内容立即切换
5. 语言选择自动保存

### 开发者使用
```typescript
// 在任何组件中使用翻译
import { useLanguage } from "@/contexts/LanguageContext";

function MyComponent() {
  const { t, language } = useLanguage();
  
  return (
    <div>
      <h1>{t('common.title')}</h1>
      <p>{t('pages.description')}</p>
    </div>
  );
}
```
