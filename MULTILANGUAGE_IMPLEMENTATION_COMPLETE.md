## 🔧 如何完成剩余页面

### 标准更新模式

每个剩余页面只需要3个步骤：

```typescript
// 步骤 1: 添加 "use client" 和导入
"use client";
import { useLanguage } from "@/contexts/LanguageContext";

// 步骤 2: 使用钩子
export default function PageName() {
  const { t } = useLanguage();
  
  // 步骤 3: 替换文本
  return (
    <div>
      <h1>{t('pages.pageTitle')}</h1>
      {/* 替换所有硬编码的中文文本 */}
    </div>
  );
}
```
