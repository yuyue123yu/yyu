## 🚀 开始部署

### 快速开始（推荐 Vercel）

#### 第一步：准备代码
```bash
# 1. 清理调试日志（可选）
# 参考 pre-deploy-cleanup.md

# 2. 提交代码
git add .
git commit -m "Ready for production"
```

#### 第二步：创建 GitHub 仓库
1. 访问 https://github.com/new
2. 创建仓库
3. 推送代码：
```bash
git remote add origin https://github.com/你的用户名/仓库名.git
git branch -M main
git push -u origin main
```

#### 第三步：准备生产数据库
1. 访问 https://supabase.com/dashboard
2. 创建新项目
3. 在 SQL Editor 中执行所有迁移脚本
4. **重要：** 执行 `create-settings-table-now.sql`

#### 第四步：部署到 Vercel
1. 访问 https://vercel.com
2. 导入 GitHub 仓库
3. 配置环境变量：
   ```
   NEXT_PUBLIC_SUPABASE_URL
   NEXT_PUBLIC_SUPABASE_ANON_KEY
   ```
4. 点击 Deploy

#### 第五步：验证
1. 访问部署的网站
2. 测试所有功能
3. 在 Admin 修改设置
4. 验证同步正常
