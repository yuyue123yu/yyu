# 法律文书模板页面测试指南

## 🌐 访问页面
打开浏览器访问：http://localhost:3000/templates

## ✅ 测试清单

### 1. Hero区域测试
- [ ] 检查渐变背景是否显示正常
- [ ] 检查背景装饰图案（模糊圆形）
- [ ] 检查动态徽章动画效果
- [ ] 测试搜索框输入功能
- [ ] 测试清除按钮（输入文字后出现X按钮）
- [ ] 检查快速统计数字是否正确显示

### 2. 分类筛选测试
- [ ] 滚动页面，检查分类栏是否固定在顶部
- [ ] 点击不同分类，检查筛选功能
- [ ] 检查"清除筛选"按钮是否出现
- [ ] 测试分类按钮的悬停效果
- [ ] 检查横向滚动是否流畅（滚动条应该隐藏）

### 3. 搜索功能测试
- [ ] 在搜索框输入关键词（如"合同"）
- [ ] 检查搜索结果是否正确过滤
- [ ] 检查搜索关键词是否显示在结果下方
- [ ] 测试清除搜索（点击X按钮）
- [ ] 测试排序选择器（虽然目前只是UI）

### 4. 模板卡片测试
- [ ] 检查卡片的悬停效果（上移、阴影）
- [ ] 检查图标的旋转动画
- [ ] 检查评分、下载数、页数显示
- [ ] 检查语言标签（🇨🇳 🇬🇧 🇲🇾）
- [ ] 检查"热门"标签（评分>=4.5的模板）
- [ ] 测试下载按钮（点击后应该生成PDF）
- [ ] 测试下载中状态（加载动画）

### 5. 模板详情弹窗测试
- [ ] 点击任意模板卡片，检查弹窗打开
- [ ] 检查弹窗的淡入和上滑动画
- [ ] 检查渐变头部和徽章
- [ ] 检查统计卡片（评分、下载数、页数）
- [ ] 检查模板信息区块（语言、文件大小、格式、更新日期）
- [ ] 检查详细描述区块
- [ ] 检查特性列表（4个勾选项）
- [ ] 测试关闭按钮（X图标）
- [ ] 测试点击背景关闭弹窗
- [ ] 测试底部下载按钮

### 6. 热门分类区域测试
- [ ] 滚动到页面底部
- [ ] 检查渐变背景效果
- [ ] 检查"热门分类"徽章
- [ ] 测试分类卡片的悬停效果
- [ ] 点击分类卡片，检查是否跳转到顶部并筛选
