## 🆘 常见问题

### Q1: 限流器重启后会清空吗？
**A:** 是的，当前使用内存存储。生产环境建议使用 Redis。

### Q2: 如何调整限流次数？
**A:** 修改 `src/lib/rateLimit.ts` 中的 `MAX_ATTEMPTS` 常量。

### Q3: 如何查看安全日志？
**A:** 检查浏览器控制台和 Vercel 日志。

### Q4: 密码策略在哪里配置？
**A:** Supabase Dashboard → Authentication → Settings
