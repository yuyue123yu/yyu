# 📋 手动测试指南

**网站地址**: https://yyu-2026.vercel.app

由于网络环境限制，请在浏览器中手动测试以下项目：

---

## 🌐 浏览器测试步骤

### 1️⃣ 基础访问测试

#### 测试首页
1. 打开浏览器（Chrome/Edge/Firefox）
2. 访问: https://yyu-2026.vercel.app
3. **检查项目**:
   - [ ] 页面能正常加载
   - [ ] 显示中文界面
   - [ ] Hero 区域显示正常
   - [ ] 导航栏显示正常
   - [ ] 页面布局正确

#### 检查浏览器控制台
1. 按 `F12` 打开开发者工具
2. 切换到 `Console` 标签
3. **检查项目**:
   - [ ] 没有红色错误信息
   - [ ] 没有 404 错误
   - [ ] 没有 CORS 错误

#### 检查网络请求
1. 在开发者工具中切换到 `Network` 标签
2. 刷新页面 (`F5`)
3. **检查项目**:
   - [ ] 所有请求状态码为 200 或 304
   - [ ] 没有失败的请求（红色）
   - [ ] 页面加载时间 < 3秒

---

### 2️⃣ 页面导航测试

#### 测试所有页面
依次访问以下页面，检查是否正常显示：

- [ ] **首页**: https://yyu-2026.vercel.app/
- [ ] **服务页面**: https://yyu-2026.vercel.app/services
- [ ] **律师页面**: https://yyu-2026.vercel.app/lawyers
- [ ] **关于我们**: https://yyu-2026.vercel.app/about
- [ ] **联系我们**: https://yyu-2026.vercel.app/contact
- [ ] **登录页面**: https://yyu-2026.vercel.app/auth/login
- [ ] **注册页面**: https://yyu-2026.vercel.app/auth/register

**每个页面检查**:
- 页面能正常加载
- 内容显示正确
- 没有控制台错误
- 布局正常（不错位）

---

### 3️⃣ API 测试

#### 测试公共 API
1. 在浏览器地址栏输入:
   ```
   https://yyu-2026.vercel.app/api/public/tenant-config
   ```

2. **预期结果**:
   - 显示 JSON 数据
   - 包含 `tenant` 对象
   - 状态码 200

3. **检查内容**:
   ```json
   {
     "tenant": {
       "id": "...",
       "name": "...",
       "subdomain": "..."
     },
     ...
   }
   ```

---

### 4️⃣ 功能测试

#### 测试语言切换
1. 在首页找到语言切换按钮
2. 点击切换语言（中文 ↔ 英文）
3. **检查项目**:
   - [ ] 页面文本切换成功
   - [ ] 刷新页面后语言保持
   - [ ] 没有控制台错误

#### 测试导航
1. 点击导航栏的各个链接
2. **检查项目**:
   - [ ] 页面跳转正常
   - [ ] URL 地址正确
   - [ ] 没有 404 错误

#### 测试响应式设计
1. 按 `F12` 打开开发者工具
2. 点击设备工具栏图标（或按 `Ctrl+Shift+M`）
3. 选择不同的设备尺寸测试
4. **检查项目**:
   - [ ] 移动端布局正常
   - [ ] 平板布局正常
   - [ ] 桌面布局正常
   - [ ] 没有内容溢出

---

#### 测试登录页面

### 7️⃣ 性能测试

#### 使用 Lighthouse 测试
1. 在 Chrome 中按 `F12` 打开开发者工具
2. 切换到 `Lighthouse` 标签
3. 选择 `Desktop` 或 `Mobile`
4. 点击 `Analyze page load`
5. **检查指标**:
   - [ ] Performance > 70
   - [ ] Accessibility > 80
   - [ ] Best Practices > 80
   - [ ] SEO > 80

#### 记录性能数据
- **First Contentful Paint (FCP)**: _____ 秒
- **Largest Contentful Paint (LCP)**: _____ 秒
- **Time to Interactive (TTI)**: _____ 秒
- **Cumulative Layout Shift (CLS)**: _____
- **Total Blocking Time (TBT)**: _____ 毫秒

---

### 8️⃣ 安全性检查

#### 检查 HTTPS
1. 查看浏览器地址栏
2. **检查项目**:
   - [ ] 显示锁图标 🔒
   - [ ] URL 以 `https://` 开头
   - [ ] 证书有效（点击锁图标查看）

#### 检查安全 Headers
1. 在开发者工具的 `Network` 标签
2. 刷新页面，点击第一个请求
3. 切换到 `Headers` 标签
4. **检查响应头**:
   - [ ] `x-frame-options`: DENY
   - [ ] `x-content-type-options`: nosniff
   - [ ] `content-security-policy`: 存在
   - [ ] `referrer-policy`: 存在

---

### 9️⃣ 错误处理测试

#### 测试 404 页面
1. 访问一个不存在的页面:
   ```
   https://yyu-2026.vercel.app/this-page-does-not-exist
   ```
2. **检查项目**:
   - [ ] 显示 404 页面
   - [ ] 页面设计友好
   - [ ] 有返回首页的链接

#### 测试 API 错误
1. 访问一个需要认证的 API:
   ```
   https://yyu-2026.vercel.app/api/tenant/users
   ```
2. **预期结果**:
   - 返回 401 Unauthorized
   - 或返回错误 JSON
