## 測試步驟

1. **清除瀏覽器緩存**
   - 按 `Ctrl + Shift + Delete`（Windows）
   - 選擇「緩存的圖片和文件」
   - 點擊「清除數據」

2. **強制刷新頁面**
   - 按 `Ctrl + Shift + R`（Windows）或 `Cmd + Shift + R`（Mac）

3. **檢查語言切換器**
   - 點擊右上角的地球圖標 🌐
   - 應該看到 4 個語言選項：
     - 🇭🇰 繁體中文 ✓（默認選中）
     - 🇨🇳 简体中文
     - 🇬🇧 English
     - 🇲🇾 Bahasa

4. **測試語言切換**
   - 點擊「简体中文」，頁面應該切換到簡體中文
   - 點擊「繁體中文」，頁面應該切換回繁體中文
   - 點擊「English」，頁面應該切換到英文
   - 點擊「Bahasa」，頁面應該切換到馬來語

## 注意事項

如果刷新後仍然顯示簡體中文：

1. **檢查 localStorage**
   - 打開瀏覽器開發者工具（F12）
   - 切換到「Application」或「存儲」標籤
   - 找到「Local Storage」→ `http://localhost:3000`
   - 查找 `language` 鍵，值應該是 `tc`
   - 如果不是，請手動刪除該鍵，然後刷新頁面

2. **清除所有緩存**
   - 使用無痕模式（Ctrl + Shift + N）測試
   - 或者完全清除瀏覽器緩存和 Cookie

3. **重啟開發服務器**
   - 停止當前的 `npm run dev`
   - 重新運行 `npm run dev`
   - 等待編譯完成後再訪問頁面
