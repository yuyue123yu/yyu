## 🔑 环境变量（准备好这些）

### 从 Supabase 获取（3 个）
访问：https://supabase.com
1. 登录 → 选择项目
2. Settings → API
3. 复制：
   - **Project URL** → `NEXT_PUBLIC_SUPABASE_URL`
   - **anon public** → `NEXT_PUBLIC_SUPABASE_ANON_KEY`
   - **service_role** → `SUPABASE_SERVICE_ROLE_KEY`

### 固定值（2 个）
- `NEXT_PUBLIC_SITE_URL` = `https://yyu.vercel.app`
- `NODE_ENV` = `production`

---

## 📋 在 Vercel 添加环境变量

在项目配置页面：
1. 找到 **"Environment Variables"**
2. 点击展开
3. 逐个添加 5 个变量：

```
Name: NEXT_PUBLIC_SUPABASE_URL
Value: （从 Supabase 复制）

Name: NEXT_PUBLIC_SUPABASE_ANON_KEY
Value: （从 Supabase 复制）

Name: SUPABASE_SERVICE_ROLE_KEY
Value: （从 Supabase 复制）

Name: NEXT_PUBLIC_SITE_URL
Value: https://yyu.vercel.app

Name: NODE_ENV
Value: production
```

4. 点击 **"Deploy"**
