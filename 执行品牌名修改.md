## 🔄 同步机制

### 方案一：使用统一配置文件（推荐）⭐⭐⭐⭐⭐

所有三个系统都从 `src/config/brand.ts` 读取品牌信息：

```typescript
// src/config/brand.ts
export const BRAND_CONFIG = {
  name: '您的品牌名',
  companyName: '您的公司全称',
  // ...
};
```

**优势：**
- ✅ 一处修改，三个系统同步更新
- ✅ 保持一致性
- ✅ 易于维护

### 方案二：通过超级管理系统设置（动态）⭐⭐⭐⭐

在超级管理系统中设置品牌信息，存储到数据库，其他系统从数据库读取。

**优势：**
- ✅ 无需修改代码
- ✅ 可以在线修改
- ✅ 支持多租户不同品牌
