## 用户流程

### 创建租户流程
1. 点击 "Create Tenant" 按钮
2. 填写基本信息（名称、子域名）
3. 配置 OEM 设置（颜色、Logo）
4. 创建管理员账号
5. 审核并确认
6. 自动跳转到租户详情

### 管理租户流程
1. 在列表中查看所有租户
2. 使用搜索和过滤器
3. 点击租户卡片查看详情
4. 编辑基本信息
5. 配置 OEM 设置
6. 激活/停用/删除租户

## 验证规则

### 基本信息
- 租户名称：必填
- 子域名：必填，仅小写字母、数字和连字符
- 自定义域名：可选

### OEM 配置
- 公司名称：必填
- 支持邮箱：必填，邮箱格式
- 颜色：十六进制格式

### 管理员账号
- 邮箱：必填，邮箱格式
- 姓名：必填

## API 集成

### 使用的端点
1. `GET /api/super-admin/tenants` - 获取租户列表
2. `POST /api/super-admin/tenants` - 创建租户
3. `GET /api/super-admin/tenants/:id` - 获取租户详情
4. `PATCH /api/super-admin/tenants/:id` - 更新租户
5. `DELETE /api/super-admin/tenants/:id` - 删除租户
6. `POST /api/super-admin/tenants/:id/activate` - 激活租户
7. `POST /api/super-admin/tenants/:id/deactivate` - 停用租户
8. `GET /api/super-admin/tenants/:id/settings` - 获取配置
9. `POST /api/super-admin/tenants/:id/settings/bulk` - 批量更新配置
