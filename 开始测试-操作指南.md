### 第一步：执行数据库迁移

1. **打开 Supabase Dashboard**
   - 访问: https://supabase.com/dashboard
   - 选择你的项目

2. **进入 SQL Editor**
   - 点击左侧菜单的 "SQL Editor"
   - 点击 "New query"

3. **执行迁移脚本**
   
   **方式 1：执行合并脚本（推荐）**
   ```sql
   -- 复制并执行这个文件的内容
   执行-账号管理系统-数据库迁移.sql
   ```
   
   **方式 2：分别执行**
   ```sql
   -- 先执行
   supabase/011_enhance_profiles_for_sub_accounts.sql
   
   -- 再执行
   supabase/012_create_password_reset_history.sql
   ```

4. **验证迁移结果**
   ```sql
   -- 执行验证脚本
   快速测试-系统状态.sql
   ```
   
   **期望看到**:
   ```
   ✅✅✅ 所有检查通过！系统已准备就绪！
   ```
