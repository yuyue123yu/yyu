# 🎯 如何修改品牌名 - 快速指南

## 🚀 三种修改方法

---

## 方法一：使用自动化脚本（最简单）⭐⭐⭐⭐⭐

### 步骤：

1. **打开终端**，进入项目目录

2. **运行脚本**：
```bash
node scripts/update-brand.js
```

3. **按提示输入**：
   - 新的品牌名（例如：法律通）
   - 新的公司全称（例如：法律通有限公司）
   - 新的邮箱域名（例如：falvtong.com）
   - 新的公司注册号（可选）

4. **确认并执行**

5. **重启开发服务器**：
```bash
npm run dev
```

### 优点：
- ✅ 自动批量替换
- ✅ 安全可靠
- ✅ 显示替换统计
- ✅ 节省时间

---

## 方法二：手动修改配置文件（推荐）⭐⭐⭐⭐

### 步骤：

1. **打开配置文件**：`src/config/brand.ts`

2. **修改品牌信息**：
```typescript
export const BRAND_CONFIG = {
  // 改这里 👇
  name: '您的品牌名',
  companyName: '您的公司全称',
  
  contact: {
    email: {
      info: 'info@yourdomain.com',  // 改这里 👇
      support: 'support@yourdomain.com',
      // ... 其他邮箱
    },
    phone: '+60 3-1234 5678',  // 改这里 👇
    address: '您的公司地址',  // 改这里 👇
  },
};
```

3. **修改核心组件**（3个文件）：

   **文件1：** `src/components/layout/Header.tsx`
   ```tsx
   <span className="text-2xl font-bold text-primary-700 tracking-tight">
     您的品牌名  {/* 改这里 */}
   </span>
   ```

   **文件2：** `src/app/admin/layout.tsx`
   ```tsx
   const [siteName, setSiteName] = useState("您的品牌名");  // 改这里
   ```

   **文件3：** `src/app/admin/login/page.tsx`
   ```tsx
   <h1 className="text-3xl font-bold text-white mb-2">您的品牌名 管理后台</h1>
   ```

4. **保存并重启**：
```bash
npm run dev
```

### 优点：
- ✅ 精确控制
- ✅ 可以逐步修改
- ✅ 适合小范围修改

---

## 方法三：全局搜索替换（快速但需谨慎）⭐⭐⭐

### 步骤：

1. **打开VS Code**

2. **按快捷键**：`Ctrl + Shift + H`（Windows）或 `Cmd + Shift + H`（Mac）

3. **输入搜索内容**：
   - 搜索：`LegalMY`
   - 替换为：`您的品牌名`

4. **预览替换**：
   - 点击每个结果查看
   - 确认是否需要替换

5. **执行替换**：
   - 点击"全部替换"或逐个替换

6. **重复以上步骤**替换邮箱域名：
   - 搜索：`legalmy.com`
   - 替换为：`yourdomain.com`

### 优点：
- ✅ 最快速
- ✅ 一次性替换所有

### 缺点：
- ⚠️ 可能替换不该替换的地方
- ⚠️ 需要仔细检查

## 🔍 如何查找所有品牌名出现的地方

### 使用VS Code搜索：
1. 按 `Ctrl + Shift + F`
2. 搜索：`LegalMY`
3. 查看所有结果
4. 逐个检查和修改

### 使用命令行（PowerShell）：
```powershell
Select-String -Path "src/**/*.tsx" -Pattern "LegalMY"
```

## ⚠️ 重要提示

### 1. 备份文件
修改前建议先备份，或使用Git：
```bash
git add .
git commit -m "备份：修改品牌名之前"
```

### 2. 邮箱域名
如果修改了品牌名，记得同时修改邮箱域名：
- `info@legalmy.com` → `info@yourbrand.com`
- `support@legalmy.com` → `support@yourbrand.com`

### 3. 公司法律信息
记得修改：
- 公司全称
- 注册号
- 公司地址

### 4. 重启服务器
修改后一定要重启开发服务器：
```bash
# 停止当前服务器（Ctrl + C）
# 然后重新启动
npm run dev
```

### 5. 清除缓存
修改后清除浏览器缓存，或使用无痕模式测试。
