## 📊 **数据统计**

### **实时统计**:
所有统计数据从 Supabase 实时读取：

```typescript
// 用户总数
const { count } = await supabase
  .from("profiles")
  .select("*", { count: "exact", head: true });

// 总收入
const { data } = await supabase
  .from("orders")
  .select("amount")
  .eq("status", "paid");
```

## 🎯 **使用流程**

### **日常管理流程**:

1. **登录后台**
   - 访问 `/admin/login`
   - 输入管理员账号

2. **查看仪表板**
   - 查看关键数据
   - 了解平台状态

3. **处理咨询**
   - 进入咨询管理
   - 确认待处理咨询
   - 分配律师

4. **管理律师**
   - 审核新律师
   - 更新律师信息
   - 控制上下线状态

5. **查看订单**
   - 监控支付状态
   - 处理退款请求

## 🆘 **常见问题**

### **Q: 无法登录管理后台？**
A: 确保已完成 Supabase 设置，并使用已注册的账号。

### **Q: 数据不显示？**
A: 检查 Supabase 数据库是否已创建表结构。

### **Q: 如何添加管理员？**
A: 在 Supabase 的 profiles 表中将 user_type 设置为 'admin'。

### **Q: 侧边栏在移动端如何使用？**
A: 点击顶部菜单按钮打开/关闭侧边栏。
