# 法律咨询网站 - 电商风格重设计

## 📋 项目概述

从电商产品经理的角度，将法律咨询网站重新设计为电商平台风格，融合京东/淘宝的成熟电商特性，提升用户体验和转化率。

---

## 🎨 设计系统

### 配色方案
- **主色**: 深蓝色 (`#1e40af`) - 专业、信任、权威
- **辅色**: 金色 (`#f59e0b`) - 高端、优质、推荐
- **中性色**: 灰色系 - 清晰、易读

### 排版风格
- **紧凑电商风格**: 充分利用空间，信息密度高
- **卡片设计**: 模块化、易扫描
- **视觉层级**: 清晰的优先级划分

## 🎨 设计亮点

1. **电商化的律师展示**: 将律师展示为"商品"，使用商品卡设计
2. **多维度筛选**: 支持价格、评分、响应时间、地区、专业领域等多维度筛选
3. **信任机制**: 通过销量、评分、评价、认证等多个维度建立用户信任
4. **促销激励**: 首次免费、优惠券、满减、积分等多种促销机制
5. **购物体验**: 购物车、收藏夹、订单总结等完整的购物流程
6. **视觉设计**: 深蓝色 + 金色配色，紧凑的电商风格排版

## 📝 使用说明

### 集成到项目
1. 所有新组件已创建在 `src/components/home/` 目录
2. 主页面 `src/app/page.tsx` 已更新，包含所有新组件
3. 所有组件都使用 Tailwind CSS 样式，无需额外配置

### 自定义
- 修改 `tailwind.config.ts` 中的颜色配置
- 修改各组件中的数据数组来自定义内容
- 修改 Tailwind 类名来调整样式

### 响应式设计
- 移动端: 2 列网格
- 平板端: 3 列网格
- 桌面端: 6 列网格
