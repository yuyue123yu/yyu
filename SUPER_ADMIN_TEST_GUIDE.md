## 🐛 常见问题

### 问题 1: 登录后立即退出
**原因**: 会话管理问题或 cookie 设置问题
**解决**: 检查浏览器 cookie 设置，清除缓存

### 问题 2: API 返回 403 错误
**原因**: 权限不足或 RLS 策略阻止
**解决**: 检查用户的 `super_admin` 标志，检查 RLS bypass

### 问题 3: 数据不显示
**原因**: API 错误或数据库连接问题
**解决**: 检查浏览器控制台，检查网络请求

### 问题 4: MFA QR 码不显示
**原因**: 库未安装或 API 错误
**解决**: 运行 `npm install otpauth qrcode`
