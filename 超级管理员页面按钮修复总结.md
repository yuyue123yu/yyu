## 🎨 统一的布局模式

### 标准布局模式

```tsx
{/* Page Header */}
<div className="space-y-4">
  {/* 标题和描述 */}
  <div>
    <h1 className="text-3xl font-bold text-gray-900">
      {t('page.title')}
    </h1>
    <p className="text-gray-600 mt-2">
      {t('page.subtitle')}
    </p>
  </div>
  
  {/* 操作按钮（如果有） */}
  <div className="flex justify-end gap-3">
    <button className="... whitespace-nowrap">
      按钮1
    </button>
    <button className="... whitespace-nowrap">
      按钮2
    </button>
  </div>
</div>
```

### 关键CSS类

1. **容器：** `space-y-4` - 垂直间距
2. **按钮组：** `flex justify-end gap-3` - 右对齐，间距3
3. **按钮：** `whitespace-nowrap` - 防止文字换行
4. **响应式：** 自动适应不同屏幕尺寸

---

## 🔧 修复原则

### 1. 垂直布局
- 标题和按钮垂直排列
- 避免水平挤压
- 更好的空间利用

### 2. 防止换行
- 添加 `whitespace-nowrap`
- 确保按钮文字完整显示
- 避免视觉混乱

### 3. 右对齐
- 按钮组右对齐
- 符合用户习惯
- 视觉平衡

### 4. 合理间距
- 标题和按钮间距：16px (`space-y-4`)
- 按钮之间间距：12px (`gap-3`)
- 统一的视觉节奏
