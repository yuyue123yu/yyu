## 🎉 部署完成后

### 更新域名（重要！）

1. 复制 Vercel 给的实际域名（如 `https://yyu-abc123.vercel.app`）
2. **Settings** → **Environment Variables**
3. 编辑 `NEXT_PUBLIC_APP_URL`，改为实际域名
4. **Deployments** → 最新部署 → **...** → **Redeploy**

## 🆘 常见问题

### 构建失败？

- 检查环境变量是否正确复制
- 查看 Vercel 构建日志

### 页面 500 错误？

- 确认 Supabase 项目正常运行
- 重新部署试试

### 找不到仓库？

- 刷新 Vercel 页面
- 确认 GitHub 已连接
