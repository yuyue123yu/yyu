# 网站设置同步说明

## 问题描述
之前在租户管理系统（Admin）修改网站设置后，前端公共网站没有实时同步显示这些修改。

## 解决方案

### 1. 创建全局设置Context
创建了 `src/contexts/SiteSettingsContext.tsx`，统一管理网站设置：

**功能特点：**
- ✅ 自动从数据库加载设置
- ✅ 提供全局访问接口
- ✅ 支持手动刷新设置
- ✅ 包含加载状态管理
- ✅ 错误处理和日志记录

**设置项包括：**
- `siteName` - 网站名称
- `siteDescription` - 网站描述
- `contactEmail` - 联系邮箱
- `contactPhone` - 联系电话
- `defaultLanguage` - 默认语言

### 2. 集成到应用布局

#### 公共网站 (`src/app/layout.tsx`)
```tsx
<SiteSettingsProvider>
  <LanguageProvider>
    <AuthProvider>
      {children}
    </AuthProvider>
  </LanguageProvider>
</SiteSettingsProvider>
```

#### 租户管理系统 (`src/app/admin/layout.tsx`)
```tsx
<SiteSettingsProvider>
  <AdminLayoutContent>{children}</AdminLayoutContent>
</SiteSettingsProvider>
```

### 3. 更新组件使用Context

#### Header组件 (`components/Header.tsx`)
```tsx
const { settings } = useSiteSettings();
// 使用 settings.siteName
```

#### Footer组件 (`components/Footer.tsx`)
```tsx
const { settings } = useSiteSettings();
// 使用 settings.siteName, settings.siteDescription, 
// settings.contactEmail, settings.contactPhone
```

#### Admin Layout (`src/app/admin/layout.tsx`)
```tsx
const { settings } = useSiteSettings();
// 使用 settings.siteName
```

  公共网站组件                              租户管理系统

## 同步机制

### 自动同步
1. **页面加载时**：Context自动从数据库加载最新设置
2. **组件挂载时**：所有使用Context的组件自动获取最新数据

### 手动刷新
在需要立即刷新的场景（如保存设置后），可以调用：
```tsx
const { refreshSettings } = useSiteSettings();
await refreshSettings();
```

### 建议优化（可选）
在 `src/app/admin/settings/page.tsx` 的保存成功后添加：
```tsx
// 保存成功后刷新Context
const { refreshSettings } = useSiteSettings();
await refreshSettings();
```

## 数据库权限

`settings` 表的RLS策略：
```sql
-- 允许所有人读取设置（包括未登录用户）
CREATE POLICY "Settings are viewable by everyone"
  ON public.settings FOR SELECT
  USING (true);

-- 只允许管理员更新设置
CREATE POLICY "Only admins can update settings"
  ON public.settings FOR UPDATE
  USING (
    EXISTS (
      SELECT 1 FROM public.profiles
      WHERE id = auth.uid() AND user_type = 'admin'
    )
  );
```

## 测试步骤

1. **修改设置**
   - 访问 `http://localhost:3000/admin/settings`
   - 修改网站名称、描述、联系方式等
   - 点击"保存设置"

2. **验证同步**
   - 刷新公共网站首页 `http://localhost:3000`
   - 检查Header中的网站名称
   - 检查Footer中的网站信息
   - 刷新Admin页面，检查侧边栏标题

3. **检查控制台**
   - 打开浏览器开发者工具
   - 查看Console日志
   - 应该看到 "✅ 网站设置已加载" 消息
