## 📝 更新模式（参考示例）

### 服务列表页更新示例

我已经完成了 `/services` 页面的更新，这是一个很好的参考示例。主要更新包括：

1. **添加 `"use client"` 指令**
2. **导入 useLanguage 钩子**
   ```typescript
   import { useLanguage } from "@/contexts/LanguageContext";
   ```

3. **在组件中使用钩子**
   ```typescript
   const { t, language } = useLanguage();
   ```

4. **替换硬编码文本**
   ```typescript
   // 之前：
   <h1>专业法律服务</h1>
   
   // 之后：
   <h1>{t('pages.servicesTitle')}</h1>
   ```

5. **处理动态内容**
   - 为数组数据创建翻译键
   - 使用 `language` 变量选择正确的翻译
   - 创建辅助函数处理复杂的翻译逻辑

## 🔧 技术细节

### 已添加的翻译键结构

```typescript
{
  common: { ... },      // 通用文本
  header: { ... },      // 头部
  footer: { ... },      // 底部
  home: { ... },        // 首页
  services: { ... },    // 服务相关
  auth: { ... },        // 认证相关
  pages: {              // 各个页面
    servicesTitle: '...',
    lawyersTitle: '...',
    consultationTitle: '...',
    templatesTitle: '...',
    knowledgeTitle: '...',
    // ... 更多页面特定的键
  }
}
```

### 翻译函数使用

```typescript
// 简单翻译
{t('common.search')}

// 嵌套键
{t('pages.servicesTitle')}

// 动态内容
{t(service.titleKey)}

// 条件翻译
{language === 'zh' ? '中文' : language === 'en' ? 'English' : 'Bahasa'}
```

1. **所有使用 `useLanguage` 的组件必须是客户端组件**
