## 💻 代码使用

```tsx
// 在任何组件中使用
import { useSiteSettings } from "@/contexts/SiteSettingsContext";

function MyComponent() {
  const { settings, loading, refreshSettings } = useSiteSettings();
  
  return <h1>{settings.siteName}</h1>;
}
```

## 📊 可用设置

```typescript
settings.siteName          // 网站名称
settings.siteDescription   // 网站描述
settings.contactEmail      // 联系邮箱
settings.contactPhone      // 联系电话
settings.defaultLanguage   // 默认语言
```
