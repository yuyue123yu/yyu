## 🎨 功能详情

### 1. 货币设置
租户可以选择计价货币：
- 马来西亚令吉 (MYR)
- 美元 (USD)
- 新加坡元 (SGD)
- 人民币 (CNY)

### 2. 咨询服务价格
三个级别的咨询服务：
- **基础咨询** - 价格、时长、描述、启用/禁用
- **标准咨询** - 价格、时长、描述、启用/禁用
- **高级咨询** - 价格、时长、描述、启用/禁用

每个服务包含：
- 价格（可自定义）
- 时长（分钟）
- 描述文字
- 启用/禁用开关

### 3. 文档服务价格
四种文档服务：
- **合同审查** - 价格、描述、启用/禁用
- **法律函件** - 价格、描述、启用/禁用
- **协议起草** - 价格、描述、启用/禁用
- **文件公证** - 价格、描述、启用/禁用

### 4. 会员套餐
可选的会员订阅服务：
- **总开关** - 启用/禁用会员功能
- **月度会员** - 价格、描述、权益列表
- **年度会员** - 价格、描述、权益列表

### 5. 优惠折扣
三种优惠活动：
- **首次咨询优惠** - 折扣比例、描述、启用/禁用
- **推荐好友优惠** - 折扣比例、描述、启用/禁用
- **批量服务优惠** - 折扣比例、描述、启用/禁用

## 🚀 使用方法

### 步骤 1: 访问价格配置页面
1. 登录租户管理后台 `/admin`
2. 点击侧边栏的"价格配置"（💰 图标）

### 步骤 2: 设置货币
选择您的计价货币（MYR, USD, SGD, CNY）

### 步骤 3: 配置咨询服务
- 设置基础/标准/高级咨询的价格
- 设置咨询时长（分钟）
- 编辑服务描述
- 使用开关启用/禁用服务

### 步骤 4: 配置文档服务
- 设置各类文档服务的价格
- 编辑服务描述
- 使用开关启用/禁用服务

### 步骤 5: 配置会员套餐（可选）
- 启用会员功能
- 设置月度/年度会员价格
- 编辑会员描述

### 步骤 6: 配置优惠折扣（可选）
- 设置折扣比例（0-100%）
- 编辑优惠描述
- 使用开关启用/禁用优惠

### 步骤 7: 保存设置
点击右上角的"保存设置"按钮

    <p className="duration">{pricing.consultation.basic.duration} 分钟</p>
    <p className="description">{pricing.consultation.basic.description}</p>
  </div>
)}

// 3. 显示文档服务价格
{pricing.documents.contract_review.enabled && (
  <div className="service-card">

    <p className="price">{pricing.currency} {pricing.documents.contract_review.price}</p>
    <p className="description">{pricing.documents.contract_review.description}</p>
  </div>
)}

// 4. 应用折扣
if (pricing.discounts.first_time.enabled && isFirstTime) {
  const discount = price * (pricing.discounts.first_time.percentage / 100);
  const finalPrice = price - discount;
}

// 5. 显示会员套餐
{pricing.membership.enabled && (
  <div className="membership-section">
